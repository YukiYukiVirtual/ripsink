※結城ゆきの使い方==================================================
run.batを実行してください。
run.batは初期状態で「java Main」となっています。設定を変更する場合は設定項目をスペースで区切って追記してください。(マイクの名前を「すごいマイク 1.0」、初期背景を赤くしたい場合の例：java Main deviceName:"すごいマイク 1.0" bgColor:ff0000)

設定はデフォルトで以下のようになっています。
{
	windowSize:256
		顔が表示されるウィンドウのサイズ(正方形)
	path:"./image"
		顔パーツを置いてあるフォルダのパス
	deviceName:null
		マイクの名前(文字列を含み、辞書順で最初に見つけた物を設定。見つからなければデフォルト)
	mouthLength:2
		口を動かす細かさ(この場合、0～1.pngが使用されます)
	mouthRate:1
		口を動かす頻度(1フレームごとにカウントダウンして0になった時に音声データ取得)
	frameRate:60
		画面更新頻度
	threshold:16000
		声を発していると認識する音量の閾値
	delay:0
		声を発してから口を動かすまでの遅延(frameRate依存)
	bgColor:00ff00
		クロマキー用背景色(16進数)
}
一部コントロールパネルで設定できる項目もあります。

※画像データの種類==================================================
base.png
	口のない顔
0.png ... (mouthLength-1).png
	口の開き具合を変えた画像
	0が一番閉じた状態

※同梱ファイル==================================================
run.bat
	実行に使用
*.class
	プログラム本体です
image/*.png
	顔のサンプルです
image/face.psd
	レイヤー分けされた顔のサンプルです
readme.txt
	このファイルです
