※結城ゆきの使い方==================================================
run.batを実行してください。
run.batは初期状態で「java Main」となっています。設定を変更する場合は設定項目をスペースで区切って追記してください。(マイクの名前を「すごいマイク 1.0」、初期背景を赤くしたい場合の例：java Main deviceName:"すごいマイク 1.0" bgColor:ff0000)
まばたきの画像を2つだけ用意し、eyeLengthを5に設定すると4フレームの間、目を閉じるという使い方ができます。また、耳の画像にすることで耳が動くようにするということもできます。

設定はデフォルトで以下のようになっています。
{
	windowSize:256
		顔が表示されるウィンドウのサイズ(正方形)
	path:"./image"
		顔パーツを置いてあるフォルダのパス
	deviceName:null
		マイクの名前(文字列を含み、辞書順で最初に見つけた物を設定。見つからなければデフォルト)
	eyeRate:30
		指定フレーム毎に1回まばたきします。
	eyeLength:2
		目を動かす細かさ(この場合、eye0～1.pngが使用されます)
		0を指定した場合は使用しません
		もし画像が存在しなければ、途中まで読み込めたものをコピーします
	mouthLength:2
		口を動かす細かさ(この場合、mouth0～1.pngが使用されます)
		0を指定した場合は使用しません
		もし画像が存在しなければ、途中まで読み込めたものをコピーします
	frameRate:30
		画面更新頻度
	threshold:16000
		声を発していると認識する音量の閾値
	delay:0
		声を発してから口を動かすまでの遅延(frameRate依存)
	bgColor:00ff00
		クロマキー用背景色(16進数)
}
一部コントロールパネルで設定できる項目もあります。

※画像データの種類==================================================
base.png
	目と口のない顔
mouth0.png ... (mouthLength-1).png
	口の開き具合を変えた画像
	0が一番「閉じた」状態
eye0.png ... (eyeLength-1).png
	目の開き具合を変えた画像
	0が一番「開いた」状態

※同梱ファイル==================================================
run.bat
	実行に使用
compile.bat
	コンパイルに使用(開発者用)
*.class
	プログラム本体です
image/*.png
	顔のサンプルです
image/face.psd
	レイヤー分けされた顔のサンプルです
readme.txt
	このファイルです

※動作環境==================================================
javaが必要です。
Windows 10、java 1.8.0_171での動作を確認しています。
